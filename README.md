# 🎮 Ultimate Rock Paper Scissors Game

An advanced and interactive Rock Paper Scissors gaming website built using HTML, CSS, and JavaScript with modern UI/UX features, authentication pages, animations, sound effects, and game systems.

---

# 🚀 Features

## 🔐 Authentication System

* Login Page
* Register Page
* Email Verification Demo
* Profile Picture Upload

---

* Paper 📄

* Water 🌊

1. Download or clone the repository
2. Open the project folder in VS Code
3. Run using Live Server
4. Register an account
5. Login and start playing 🎮
